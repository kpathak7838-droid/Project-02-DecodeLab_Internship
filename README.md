# Project-02-DecodeLab_Internship
This project was developed as part of the DecodeLabs Full Stack Development Internship Program. The objective of Project 02 is to develop a simple backend API that handles the users requests, input data, and returns JSON responses.

Objective
Develop a simple backend API that handles user requests, validates input data, and returns JSON responses.

Features
GET endpoint to fetch student data
POST endpoint to add new students
JSON data handling
Input validation
Express.js server setup
Technologies Used
Node.js
Express.js
CORS
JavaScript

API Endpoints
GET
GET /students

Returns all student records.

POST
POST /students

Adds a new student record.
